# Finance Tracker

Finance Tracker is a web application that allows you to track your income and expenses. It provides a user-friendly interface to add transactions and gives you an overview of your financial situation.

## Features

- Add transactions with descriptions, amounts, and categories (income or expense).
- Real-time updates of your balance, income, and expenses.
- Transaction history with descriptions and amounts for income and expenses.
- Interactive and responsive design for an enhanced user experience.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository:

```
git clone https://github.com/Tabhi109/TechGig-JS-finance-tracker.git
```

2. Open the index.html file in your preferred web browser.

## Usage

1. Enter a description for your transaction in the "Description" field.
2. Select the appropriate radio button for income or expense.
3. Enter the transaction amount in the "Amount" field.
4. Click the "Add" button to add the transaction.
5. The balance, income, and expense values will update accordingly, and the transaction will be listed in the respective income or expense section.
